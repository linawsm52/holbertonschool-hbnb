# HBnB Evolution

## Project Overview
HBnB Evolution is a simplified AirBnB-like application designed to demonstrate the architecture, design, and implementation of a multi-layered software system. The project is divided into multiple parts, each focusing on a specific phase of the software development lifecycle.


## Project Objectives
- Design a scalable and well-structured application architecture
- Use UML diagrams to document system design
- Prepare blueprint for future implementation phases

## Project Structure
The project is organized into the following parts:

### Part 1: Technical Documentation
Focuses on documenting the architecture and design of the HBnB Evolution application, including:

- UML class diagrams
- UML sequence diagrams for core API flows
